CONFIGURATION:
===============

Set planner type via environment variable:
  export BATTERY_PLANNER_TYPE=heuristic    # (default)

Or select in code:
  planner = BatteryPlannerFactory.create('heuristic')

ADDING NEW PLANNERS:
====================

1. Create battery_planners/my_planner.py:
   
   from battery_planners import BatteryPlanner, BatteryPlanEntry
   
   class MyPlannerPlanner(BatteryPlanner):
       def plan(self, predictions_kwh, solar_kwh, import_prices, ...):
           entries = []
           for i in range(len(predictions_kwh)):
               entry = BatteryPlanEntry(...)
               entries.append(entry)
           return entries

2. Register in battery_planners/factory.py:
   
   from .my_planner import MyPlannerPlanner
   BatteryPlannerFactory.register('my_planner', MyPlannerPlanner)

3. Use:
   planner = BatteryPlannerFactory.create('my_planner')
   plan = planner.plan(predictions, solar, prices, timestamps)
